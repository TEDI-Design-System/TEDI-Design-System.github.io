# TEDI Design System Landing Page

This repository hosts the landing page for the TEDI Design System, providing quick access to both React and Angular Storybook documentation.

## About TEDI Design System

TEDI-Ready is a comprehensive design system that provides production-ready components for modern web applications. It includes:

- **Extensive Component Library**: Buttons, forms, navigation, overlays, notifications, and more
- **Dual Framework Support**: Available for both React and Angular
- **Accessibility First**: Components built with WCAG standards in mind
- **Consistent Design**: Uses TEDI brand colors and design principles
- **Developer Friendly**: Well-documented components with live examples

## Technical Details

The landing page is built with pure HTML and CSS, featuring:

- TEDI brand colors (#005AA3, #004277)
- TEDI component styling (buttons, cards)
- Official React and Angular SVG logos
- Responsive design for all screen sizes
- No external dependencies
